Для того что бы запустить данный сервис необходимо следующее:
1) Подключить базу данных - PostgreSQL. Настройки необходимо заполнить в файле application.properties
    src/main/resources/application.properties - путь к настройкам
    необходимо прописать следующие параметры -
    spring.dataSource.url=jdbc:postgresql://...
    spring.dataSource.username=...
    spring.dataSource.password=...

2) Для того что бы реализовать импорт дампа данных из файла JSON в базу данных, необходимо
    поместить JSON файл в пакет - src/main/resources/static/ и назвать его input.json
    отправить post запрос на энедпойнт - /api/admin/import

3) Для получения данных из базы данных нужно -
    Для того что бы получить форматированный ответ - необходимо отправить get запрос по эндпойнту - /wiki/<название статьи>/?pretty=true
    Для того что бы получить ответ в виде строки, флаг pretty должен быть равен false - /wiki/<название статьи>/?pretty=false
